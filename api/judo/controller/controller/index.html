update_traces() → None[source]¶

Update traces by extracting data from sensors readings.

We need to have num_spline_points - 1 line segments. Sensors will initially be of shape (num_rollout x num_timesteps x nsensordata) and needs to end up being in shape (num_elite * num_trace_sensors * size of a single rollout x 2 (first and last point of spline) x 3 (3d pos))

Return type:

None
